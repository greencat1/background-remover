## Используемая модель

В качестве основной модели используется BiRefNet.

Причины выбора:

- высокое качество выделения объектов;
- корректная обработка мелких деталей;
- качественное выделение волос и шерсти;
- устойчивость к сложному фону;
- современные результаты среди моделей удаления фона.

Этапы обработки изображения:

1. Загрузка изображения.
2. Предобработка:
   - изменение размера;
   - нормализация;
   - преобразование в тензор.
3. Инференс модели BiRefNet.
4. Получение маски объекта.
5. Постобработка:
   - изменение размера маски;
   - создание альфа-канала;
   - генерация итогового PNG.

Общий поток:

```text
Изображение
      ↓
Предобработка
      ↓
BiRefNet
      ↓
Постобработка
      ↓
PNG с прозрачным фоном
```

## Установка зависимостей

Создать виртуальное окружение:

```bash
python -m venv venv
```

Активировать окружение:

Windows:

```bash
venv\Scripts\activate
```

Linux:

```bash
source venv/bin/activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

## Использование API

Метод:

```text
POST /remove-bg
```

Параметр:

```text
file — изображение
```

Пример запроса через curl:

```bash
curl -X POST \
-F "file=@images/input/cat.jpg" \
http://localhost:8000/remove-bg \
--output result.png
```

## Логирование

Во время обработки сервис выводит этапы выполнения:

Пример:

```text
========================

[REQUEST] New image received

[LOAD] Image size: (548, 364)
[LOAD] Time: 0.01s

[PREPROCESS] Tensor shape: torch.Size([1,3,384,384])
[PREPROCESS] Time: 0.00s

[INFERENCE] Running model...
[INFERENCE] Time: 3.14s

[POSTPROCESS] Processing mask...
[POSTPROCESS] Time: 0.12s

[DONE] Total time: 3.27s
```

Логи позволяют отслеживать время работы каждого этапа и искать узкие места.
